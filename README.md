## 1️⃣ E-Commerce Sales Analysis

### Business Problem

An e-commerce company wants to understand:

* Which product categories generate the most revenue
* Which cities contribute the highest order volume
* Which products are profitable versus loss-making
* How sales fluctuate over time

### Approach

Performed exploratory and analytical SQL querying using:

* Aggregate Functions
* GROUP BY
* ORDER BY
* CASE Statements

### Key Business Insights

* Revenue-generating product categories identified
* Top-performing cities by order volume
* Monthly sales trends analyzed
* Most profitable products highlighted
* Loss-making products detected

### SQL Concepts Used

```sql
GROUP BY
ORDER BY
SUM()
COUNT()
AVG()
CASE
```

### Skills Demonstrated

* Data Analysis
* Revenue Analytics
* Business Reporting
* KPI Development

## 2️⃣ BigBasket Product Performance Analysis

### Business Problem

Analyze product pricing, discounts, ratings, and category performance to support:

* Pricing strategy
* Product positioning
* Customer preference analysis
* Product portfolio optimization

### Approach

Utilized advanced SQL techniques to create reusable analytical solutions.

### Key Business Insights

* Product rankings using Window Functions
* Price trend analysis using Lead/Lag Functions
* Category-wise top-rated products
* Discount analysis through Views
* Dynamic reporting using Stored Procedures

### SQL Concepts Used

```sql
CTEs
Window Functions

DENSE_RANK()

Views
```
